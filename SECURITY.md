# Security Policy

## Supported Versions

This project is a static website served from the `main` branch. Only the
current state of `main` is supported; there are no versioned releases.

## Scope

In scope:

- The static site in this repository (HTML, CSS, JavaScript, and JSON data).
- The Cloudflare Pages configuration that serves it.

Out of scope:

- The third-party organization websites linked from the resource list. Please
  report issues with those sites directly to the organizations that own them.

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Use GitHub's private vulnerability reporting (the **Report a vulnerability**
button on the repository **Security** tab), or email
`security@phantomtechindustries.net`.

Please include enough detail to reproduce the issue. We aim to acknowledge
reports within 72 hours and will keep you informed as we work toward a fix.

## No Sensitive Data

This site stores only public information: organization names, public mission
statements, public website URLs, and public crisis hotline numbers. It does
not collect or store any personal data about visitors or veterans.
